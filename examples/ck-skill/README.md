# ck-skill

Demonstrates creating a comprehensive skill that teaches an agent how to use `ck` (semantic code search).

This example showcases how to build detailed, structured skills with multiple sections, setup instructions, usage examples, and workflow guidance. The generated skill provides an agent with everything needed to effectively use semantic search for code discovery.

## Files

- `ck-skill.fsx` - Script that generates the skill definition
- `ck-semantic-search.SKILL.md` - Output skill file with search modes, tuning guidance, and workflows
