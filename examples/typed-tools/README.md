# typed-tools

Demonstrates type-safe tool configuration and harness-specific output mapping.

This example shows how to use typed tools (Write, Edit, Bash, WebFetch, etc.) with full IDE autocomplete, disable tools with `disallowedTools`, and render the same agent definition for different harnesses (Opencode, ClaudeCode, Copilot). It illustrates how tool names are mapped based on platform conventions (lowercase vs. capitalized).
