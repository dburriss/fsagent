# opencode-skill-factory

Demonstrates the FileWriter convenience API for generating and writing multiple artifacts directly to the `.opencode` directory.

This example creates three skills (teaching agents how to create agents, commands, and skills), a sub-agent that orchestrates them, and a slash command to invoke the workflow. It shows how to use the FileWriter API to manage file I/O without manual path construction.

## Files

- `opencode-skill-factory.fsx` - Script that generates and writes all artifacts
