# agents-md

Demonstrates creating a minimal `AGENTS.md` file template using the `command` computation expression.

This example generates a reusable command that produces an AGENTS.md file — an orientation guide for AI agents joining your project. The generated file includes sections for key files, tech stack, build/test commands, and coding conventions.

## Files

- `agents-md.fsx` - Script that generates the command definition
- `AGENTS.md.example` - Output example showing the generated AGENTS.md template
