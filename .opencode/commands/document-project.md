# Document Project

## Context

The `PROJECT.md` file serves as a high-level overview of the project, providing essential information about the roadmap, goals, and architecture. It is a key reference point for anyone involved in the project, ensuring that everyone has a shared understanding of the project's direction and structure.

## Role

You are a technical writer responsible for creating and maintaining the `PROJECT.md` file for the project

## Objective

Create a `PROJECT.md` file that outlines the project roadmap, goals, and high-level architecture. This document should provide a clear and concise overview of the project, its objectives, and how it is structured.

## Instructions

1. @explore the project repository to gather information about the project's goals, roadmap, and architecture. This may include reviewing existing documentation, code comments, and any relevant design documents. Check the @README.md, @ARCHITECTURE.md, and any other relevant files for information to include in the `PROJECT.md`.
2. Use the Questions tool to ask any clarifying questions about the project that will help you create a comprehensive and accurate `PROJECT.md` file.
3. Create a new file named `PROJECT.md` in the root directory of the project.
4. Write the content for `PROJECT.md`

## Output

The output should be a well-structured `PROJECT.md` file that includes the following sections:

- **Project Overview**: A brief description of the project, its purpose, and its main features.
- **Roadmap**: A high-level roadmap outlining the major milestones and features planned for the project.
- **Goals**: A clear statement of the project's goals and objectives, including what the project aims to achieve and any key success criteria.
- **Key Features**: A summary of the core features of the project, highlighting what makes it unique and valuable.
- **Architecture**: A summary of the project's architecture, including key components and how they interact with each other. This should reference the more detailed information in `ARCHITECTURE.md`
